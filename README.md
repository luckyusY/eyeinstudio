# Eye in Studio

Premium photography, printing, framing and studio operations platform for Eye in Studio, Remera, Kigali.

## Local development

1. Copy `.env.example` to `.env.local` and provide MongoDB and Cloudinary credentials.
2. Run `npm install`.
3. Run `npm run dev` and open `http://localhost:3000`.

The first release includes the public marketing experience, portfolio, service packages, booking requests, print uploads/orders, SEO metadata and a database-backed admin snapshot.
